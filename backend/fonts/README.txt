В этой директории нужно разместить шрифт для прописей по методике В. Илюхиной.

Имя файла должно быть: iluhina.ttf или iluhina.otf

После добавления шрифта, раскомментируйте следующие строки в файле backend/main.py:

```python
# font_path = "fonts/iluhina.ttf"
# pdfmetrics.registerFont(TTFont('Iluhina', font_path))
```

И измените код генерации PDF для использования этого шрифта.

Примечание: Если шрифт является коммерческим, убедитесь, что у вас есть соответствующая лицензия на его использование.
